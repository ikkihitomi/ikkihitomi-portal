一箕CMS Ver1.00

2026年7月

概要

一箕地区ポータルの管理者向け運営システムとして、 「一箕CMS Ver1.00」を構築しました。 記事の作成・編集・削除、カテゴリ管理、公開状態管理、 アイキャッチ画像の登録まで対応しました。

主な機能

  • Supabase Authentication による管理者ログイン
  • 管理者権限管理
  • 記事の新規作成
  • 記事の編集・更新
  • 記事の削除
  • カテゴリ選択
  • 下書き・公開状態の管理
  • Supabase Storage によるアイキャッチ画像アップロード
  • 記事一覧でのサムネイル表示

技術基盤

  • GitHub によるバージョン管理
  • Vercel による公開環境
  • Supabase Database
  • Supabase Authentication
  • Supabase Storage
